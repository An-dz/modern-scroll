----- Cette extension nécessite d'avoir Opera 12.10 ou plus récent d'installé ! -----
Modern scroll porte le défilement à l'écran à un tout nouveau niveau
(Les barres de défilement standards d'Opera doivent être désactivées pour éviter la présence de deux barres. Pour cela aller dans le menu Opera → Réglages → Préférences... (Ctrl+F12) → Avancé → Navigation pour désactiver "Afficher les barres de défilement"

Qu'est ce que Modern scroll apporte-il de génial ?

+ il augmente la performance du défilement !
+ la vitesse de défilement est réglable
+ ses barres de défilement ne prennent pas plus de place, donc toute la largeur de l'écran peut être utilisée pour afficher les pages web
+ les éléments ou les pages entières ne bougent pas d'un pixel lorsque le des pages web change ou que les bars deviennent (non-)nécessaires
+ les barres de défilement sont également utilisables en mode plein-écran
+ les barres de défilement se masquent automatiquement lorsqu'elles ne sont pas nécessaires
+ une "barre des tâches" unique, rendant possible le défilement à la fois horizontal et vertical
+ des boutons optionnels pour faire défiler directement vers le haut ou le bas de la page
+ le comportement et l'apparence de la barre sont totalement personnalisables

Cette extension était initialement conçue pour succéder à "Slim ScrollBar" de damage92. Mais puisqu'elle n'avait pas la moindre ligne de code de programmation commune avec l'extension originale, nous avons décidé de la publier sous la forme d'une nouvelle extension. Slim ScrollBar va bien sûr continuer d'exister si vous la préférez ou si vous utilisez Opera dont la version est inférieure à la 12.10, sous laquelle Modern scroll n'est pas disponible. D'autant plus qu'en raison du peu de temps libre dont dispose damage92, son développement actif pour les versions inférieures n'est pas sa priorité.
